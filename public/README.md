# Public Folder

Deze folder bevat statische assets die publiekelijk toegankelijk zijn.

## Gebruik

Plaats hier bestanden zoals:
- Logo's (bijv. `woc-logo.png`, `woc-logo.svg`)
- Favicon (`favicon.ico`)
- Andere statische afbeeldingen

## Toegang

Bestanden in deze folder zijn toegankelijk via de root URL:
- `/woc-logo.png` → `public/woc-logo.png`
- `/favicon.ico` → `public/favicon.ico`

## WOC Logo

Plaats je WOC logo bestand hier met een duidelijke naam zoals:
- `woc-logo.png`
- `woc-logo.svg` (aanbevolen voor schaalbaarheid)
